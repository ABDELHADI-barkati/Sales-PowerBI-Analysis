--Beverages Sales Analysis (2021-2023)
--Data Analyst Portfolio Project--

--Project Overview
This project analyzes a massive dataset of over --890,000 records--sourced from Kaggle. The goal was to transform raw sales data into actionable insights using a professional data pipeline.

-- Technical Stack
--Storage:-- SQL Server (Data ingestion & structured storage).
--ETL:-- Power Query (Data cleaning, filtering 2021-2023, and accuracy checks).
--Visualization:--Power BI (Interactive dashboards and trend analysis).

-- Key Insights
--Sales Leader:- Alcoholic beverages consistently dominated the market share throughout the period.
--Top Performance:-2023-- emerged as the peak year with record-breaking revenue and order volume.
-Efficiency: Optimized a 1M row dataset into a high-performance 2-page dashboard.
